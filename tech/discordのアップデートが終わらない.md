Webブラウジングやネットゲームをプレイするのは何にも問題がないのに、Discordのアップデートだけ異常に遅いことがあった。
3時間くらい放置しててもアップデートが終わらず、PCを再起動しても同じ状態。

Discordが異常な状態になってしまったのかと思い、一度アンインストールして公式サイトからクライアントをダウンロードすると、なんと500kB/sしか出ていない。

これはおかしいと思いx.comで検索をかけて見るが何もヒットしない。どうやらうちだけの問題らしい。

難しいことはわからないが、おそらくうちのルータからDiscordのサーバのネットワーク間で問題が起きているらしい事はわかる。
ぱっと思いついてまず試せるのはルータの再起動だろう。でも面倒だったのでできなかった。
代わりにDNSキャッシュを削除してみた。

```powershell
% ipconfig /flushdns

Windows IP Configuration

Successfully flushed the DNS Resolver Cache.
```

上記のコマンドを実行後、Discordを起動してみると一瞬でアップデートが完了し正常に起動した。
なぜこれで改善するのかわからない。こういうこともあるんだなくらいで覚えておくことにする。
